# Koa Starter

This is a simple Koa web app starter.

## Stack
- Node.js(8)
- Koa.js(async/await)
- Postgresql Connection Pool(pg, bluebird)
- Env Import(dotenv)
- Logging(chalk)
- Syntax(eslint)
- Custom build path(jsconfig.json)
- Real time watching(nodemon)
